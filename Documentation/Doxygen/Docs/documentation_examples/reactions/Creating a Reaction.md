Creating a Reaction {#creating_a_reaction}
============
- Execute the, `rtc_rest::reactions::createReactionAsync()` function, while passing in a data structure of type `rtc_rest::create_reaction_data`, with a return value of type `auto`, or, `rtc_rest::reaction`.
- call the function with `rtc_rest::co_routine::get()` added to the end in order to wait for its return value now.
